### 18-2학기 프로젝트 - 보험사 시스템 개발
***
    개발 환경: Windows 10 64 bit, IntelliJ IDEA
    활용 기술: Spring Framework(JDBC)
    
    데이터베이스: MySQL 8.0.xx
    
* DDL.txt: 테이블 생성 쿼리  

* src/main/**: 실행 메뉴  

* src/global/**: 공통 기능  

* src/dao/**: DB 액세스 관련  

* src/client/**: 고객 및 고객 소유 자산 관련  

* src/contract/**: 보험 계약, 보험료 납부, 보험금 청구 관련  

* src/insurance/**: 각종 보험상품 관련
